# Application de gestion d’un cabinet médical

## Présentation du projet

Cette application est un système de gestion d’un cabinet médical développé avec **Java** et **SQL**.
Elle a pour objectif de faciliter l’organisation quotidienne d’un cabinet médical à travers une interface simple permettant de gérer les principales informations liées aux patients, aux rendez-vous, aux factures, aux ordonnances, aux dossiers médicaux et aux consultations.

Le projet intègre également un système d’authentification avec gestion des rôles, permettant de différencier les accès entre un **assistant** et un **médecin**.

## Fonctionnalités principales

* Authentification des utilisateurs
* Gestion des rôles : Assistant / Médecin
* Gestion des patients
* Gestion des rendez-vous
* Gestion des factures
* Gestion des ordonnances
* Gestion des dossiers médicaux
* Gestion des consultations
* Ajout, modification et suppression des données
* Connexion à une base de données SQL

## Technologies utilisées

* Java
* SQL
* Java Swing
* JDBC
* NetBeans
* Base de données relationnelle

## Objectif du projet

L’objectif de ce projet est de concevoir une application de gestion simple, claire et fonctionnelle, adaptée aux besoins de base d’un cabinet médical.

Ce projet m’a permis de renforcer mes compétences en développement Java, en conception d’interfaces graphiques avec Java Swing, en manipulation de bases de données SQL et en connexion entre une application et une base de données à l’aide de JDBC.

## Installation et utilisation

1. Cloner le projet :

```bash
git clone https://github.com/Arezkiloun/Gestion_Cabinet_Medical.git
```

2. Ouvrir le projet avec **NetBeans**.

3. Configurer la base de données SQL selon les paramètres du projet.

4. Vérifier la connexion JDBC.

5. Lancer l’application depuis NetBeans.
